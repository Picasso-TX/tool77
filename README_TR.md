# tr Tools

Tool77 (tool77.com), çevrimiçi araç toplama sitesidir (hepsi bir arada platform), temel konsepti:👉 “Web sayfasını aç ve araçları anında kullan, kurulum veya kayıt gerekmez.”

# 🧰 Tool77 nedir?
Tool77, tarayıcı tabanlı ücretsiz bir araç kutusudur ve birçok günlük ve geliştirici aracı sunar:
📄 Belge araçları (PDF / Word / PPT / HTML dönüşümü)
🎬 Video indirme (YouTube, TikTok, Instagram vb.)
🖼️ Görsel işleme (format dönüştürme, ICO oluşturucu vb.)
🧑‍💻 Geliştirici araçları (JSON, HTTP, regex vb.)
🌐 Web araçları (URL kodlama, HTML dönüşümü vb.)
🕒 Zaman ve hesaplama araçları (birim dönüşümü, tarih araçları vb.)
🔐 Güvenlik araçları (hassas kelime tespiti vb.)

👉 Özetle, “çevrimiçi verimlilik araç kutusu”dur.
