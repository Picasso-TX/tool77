# vi Tools

Tool77 (tool77.com) là website tổng hợp công cụ trực tuyến (nền tảng all-in-one), với ý tưởng chính:👉 “Mở trang web và sử dụng công cụ ngay, không cần cài đặt hay đăng ký.”

# 🧰 Tool77 là gì?
Tool77 là một bộ công cụ miễn phí chạy trên trình duyệt, cung cấp nhiều công cụ hàng ngày và cho lập trình viên, như:
📄 Công cụ tài liệu (PDF / Word / PPT / HTML)
🎬 Tải video (YouTube, TikTok, Instagram, v.v.)
🖼️ Xử lý hình ảnh (chuyển đổi định dạng, tạo ICO, v.v.)
🧑‍💻 Công cụ lập trình (JSON, HTTP, regex, v.v.)
🌐 Công cụ web (mã hóa URL, chuyển đổi HTML, v.v.)
🕒 Công cụ thời gian & tính toán (đổi đơn vị, ngày tháng, v.v.)
🔐 Công cụ bảo mật (phát hiện từ nhạy cảm, v.v.)

👉 Tóm lại, đây là “bộ công cụ năng suất trực tuyến”.
