# 🧰 What is Tool77?
Tool77 is a free browser-based toolbox website that provides a wide range of daily and developer tools, such as:
📄 Document tools (PDF / Word / PPT / HTML conversion)
🎬 Video download tools (YouTube, TikTok, Instagram, etc.)
🖼️ Image processing tools (format conversion, ICO generator, etc.)
🧑‍💻 Developer tools (JSON formatter, HTTP tools, regex tools, etc.)
🌐 Web tools (URL encoding, HTML conversion, etc.)
🕒 Time & calculation tools (unit conversion, date tools, etc.)
🔐 Security tools (sensitive word detection, etc.)

👉 In essence, it is a “web-based tool collection + online productivity toolbox.”
