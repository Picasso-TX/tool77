# ja Tools

Tool77（tool77.com）はオンラインツール集約サイト（オールインワンプラットフォーム）で、核心的なコンセプトは：👉「Webページを開くだけでツールをすぐ使える、インストールや登録は不要」

# 🧰 Tool77とは？
Tool77はブラウザベースの無料ツールボックスで、日常や開発向けの多くのツールを提供しています：
📄 ドキュメントツール（PDF / Word / PPT / HTML変換）
🎬 動画ダウンロード（YouTube、TikTok、Instagramなど）
🖼️ 画像処理（形式変換、ICO生成など）
🧑‍💻 開発ツール（JSON整形、HTTPツール、正規表現など）
🌐 Webツール（URLエンコード、HTML変換など）
🕒 時間・計算ツール（単位変換、日付ツールなど）
🔐 セキュリティツール（センシティブワード検出など）

👉 本質的には「オンライン効率ツールボックス」です。
