# ms Tools

Tool77 (tool77.com) ialah laman agregator alat dalam talian (platform semua-dalam-satu), dengan konsep utama:👉 “Buka laman web dan gunakan alat serta-merta tanpa pemasangan atau pendaftaran.”

# 🧰 Apa itu Tool77?
Tool77 ialah kotak alat percuma berasaskan pelayar yang menyediakan pelbagai alat harian dan pembangun, seperti:
📄 Alat dokumen (PDF / Word / PPT / HTML)
🎬 Muat turun video (YouTube, TikTok, Instagram, dll.)
🖼️ Pemprosesan imej (penukaran format, penjana ICO, dll.)
🧑‍💻 Alat pembangun (JSON, HTTP, regex, dll.)
🌐 Alat web (pengekodan URL, penukaran HTML, dll.)
🕒 Alat masa & pengiraan (penukaran unit, tarikh, dll.)
🔐 Alat keselamatan (pengesanan perkataan sensitif, dll.)

👉 Ringkasnya, ia ialah “kotak alat produktiviti dalam talian”.
