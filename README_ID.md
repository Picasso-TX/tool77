# id Tools

Tool77 (tool77.com) adalah situs agregator alat online (platform all-in-one), dengan konsep utama:👉 “Buka website dan langsung gunakan alat tanpa instalasi atau registrasi.”

# 🧰 Apa itu Tool77?
Tool77 adalah toolbox gratis berbasis browser yang menyediakan berbagai alat harian dan developer, seperti:
📄 Alat dokumen (PDF / Word / PPT / HTML)
🎬 Unduh video (YouTube, TikTok, Instagram, dll.)
🖼️ Pengolahan gambar (konversi format, generator ICO, dll.)
🧑‍💻 Alat developer (JSON, HTTP, regex, dll.)
🌐 Alat web (encoding URL, konversi HTML, dll.)
🕒 Alat waktu & kalkulasi (konversi unit, tanggal, dll.)
🔐 Alat keamanan (deteksi kata sensitif, dll.)

👉 Intinya, ini adalah “toolbox produktivitas online”.
