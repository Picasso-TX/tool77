# fr Tools

Tool77 (tool77.com) est un site de regroupement d’outils en ligne (plateforme tout-en-un), avec pour objectif principal :👉 “Ouvrir la page web et utiliser des outils instantanément, sans installation ni inscription.”

# 🧰 Qu’est-ce que Tool77 ?
Tool77 est une boîte à outils gratuite basée sur le navigateur qui propose de nombreux outils du quotidien et pour développeurs, comme :
📄 Outils de documents (PDF / Word / PPT / HTML)
🎬 Téléchargement de vidéos (YouTube, TikTok, Instagram, etc.)
🖼️ Traitement d’images (conversion de formats, génération ICO, etc.)
🧑‍💻 Outils développeur (JSON, HTTP, expressions régulières, etc.)
🌐 Outils web (encodage URL, conversion HTML, etc.)
🕒 Outils de temps et calcul (unités, dates, etc.)
🔐 Outils de sécurité (détection de mots sensibles, etc.)

👉 En bref, une “boîte à outils en ligne pour la productivité”.
