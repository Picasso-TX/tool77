# pt Tools

Tool77 (tool77.com) é um site de agregação de ferramentas online (plataforma tudo-em-um), com o objetivo principal:👉 “Abrir o site e usar ferramentas instantaneamente, sem instalação ou registro.”

# 🧰 O que é o Tool77?
Tool77 é uma caixa de ferramentas gratuita baseada no navegador que oferece várias ferramentas do dia a dia e para desenvolvedores, como:
📄 Ferramentas de documentos (PDF / Word / PPT / HTML)
🎬 Download de vídeos (YouTube, TikTok, Instagram, etc.)
🖼️ Processamento de imagens (conversão de formatos, gerador ICO, etc.)
🧑‍💻 Ferramentas para desenvolvedores (JSON, HTTP, regex, etc.)
🌐 Ferramentas web (codificação de URL, conversão HTML, etc.)
🕒 Ferramentas de tempo e cálculo (conversões, datas, etc.)
🔐 Ferramentas de segurança (detecção de palavras sensíveis, etc.)

👉 Em resumo, uma “caixa de ferramentas online de produtividade”.
